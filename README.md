## üìñ User Guide

### 1. Paper Search

Search millions of papers from arXiv database.

**How to use:**
1. Enter keywords in the search box (e.g., "transformer attention mechanism")
2. Press Enter or click the search button
3. Browse results with paper title, authors, abstract preview
4. Click on a paper to view details and PDF

**Search tips:**
- Use specific keywords for better results
- Search by arXiv ID directly (e.g., "2301.00001")
- Combine multiple keywords for precise matching

---

### 2. PDF Upload & Analysis

Upload your own PDF papers for analysis.

**Supported formats:** PDF

**How to use:**
1. Click the upload button or drag & drop a PDF file
2. Wait for the file to be processed
3. The PDF viewer will open automatically
4. Use AI features to analyze the content

---

### 3. Data Visualization

Transform paper data into beautiful, interactive charts.

**Available Templates:**

| Category | Templates |
|----------|-----------|
| **Basic Charts** | Bar, Line, Pie, Area, Scatter |
| **Statistical** | Box Plot, Violin Plot, Histogram, Ridgeline |
| **Hierarchical** | Treemap, Sunburst, Tree Chart |
| **Network** | Citation Network, Research Cluster, Density Network |
| **3D Charts** | 3D Bar, 3D Scatter, 3D Pie |
| **Scientific** | Heatmap, GO Term, Survival Curve, UpSet Plot |
| **Flow** | Sankey Diagram, Funnel Chart |
| **Text** | Word Cloud |

**How to use:**
1. Open a paper or upload a PDF
2. Click on a visualization template from the sidebar
3. The AI will automatically extract relevant data
4. Customize the visualization using the chat interface
5. Download or share your visualization

---

### 4. AI Chat

Chat with AI to analyze papers and modify visualizations.

**Features:**
- Ask questions about paper content
- Translate sections to different languages
- Modify generated visualizations with natural language
- Get explanations of complex concepts

**How to use:**
1. Open a paper or visualization
2. Type your question or request in the chat box
3. AI will respond based on the current context

**Example prompts:**
- "Summarize the main contributions of this paper"
- "Translate the abstract to Chinese"
- "Change the chart colors to blue theme"
- "Add a title to the visualization"

---

### 5. My Space

Manage your saved papers and visualizations.

**Features:**
- View history of searched papers
- Access uploaded PDFs
- Browse generated visualizations
- Filter by time range
